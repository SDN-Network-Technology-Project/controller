StatisticsNorthbound

Northbound APIs that returns various Statistics exposed by the Southbound protocol plugins such as Openflow.

Authentication scheme : HTTP Basic
Authentication realm : opendaylight
Transport : HTTP and HTTPS

HTTPS Authentication is disabled by default.

The following resources are part of this group:

  • /{containerName}/flow
  • /{containerName}/port
  • /{containerName}/table
  • /{containerName}/flow/node/{nodeType}/{nodeId}
  • /{containerName}/port/node/{nodeType}/{nodeId}
  • /{containerName}/table/node/{nodeType}/{nodeId}
