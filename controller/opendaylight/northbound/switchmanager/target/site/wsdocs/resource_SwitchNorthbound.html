SwitchNorthbound

The class provides Northbound REST APIs to access the nodes, node connectors and their properties.

The following resources are part of this group:

  • /{containerName}/nodes
  • /{containerName}/save
  • /{containerName}/node/{nodeType}/{nodeId}
  • /{containerName}/node/{nodeType}/{nodeId}/property/{propertyName}
  • /{containerName}/node/{nodeType}/{nodeId}/property/{propertyName}/{propertyValue}
  • /{containerName}/nodeconnector/{nodeType}/{nodeId}/{nodeConnectorType}/{nodeConnectorId}/property/{propertyName}
  • /{containerName}/nodeconnector/{nodeType}/{nodeId}/{nodeConnectorType}/{nodeConnectorId}/property/{propertyName}/{propertyValue}

/{containerName}/nodes

Mount Point: /controller/nb/v2/switchmanager/{containerName}/nodes

GET

Retrieve a list of all the nodes and their properties in the network

Parameters

name description type default
containerName Name of the Container (Eg. 'default') path

Response Body

element: list
media types: application/xml
application/json

A list of Pair each pair represents a org.opendaylight.controller.sal.core.Node and Set of org.opendaylight.controller.sal.core.Property attached to it.

http://localhost:8080/controller/nb/v2/switchmanager/default/nodes
