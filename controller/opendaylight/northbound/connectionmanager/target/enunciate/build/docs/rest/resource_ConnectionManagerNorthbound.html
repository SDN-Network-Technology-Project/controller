/nodes

Mount Point: /controller/nb/v2/connectionmanager/nodes

GET

Retrieve a list of all the nodes connected to a given controller in the cluster.

Parameters

name description type default
controller Optional parameter to retrieve the nodes connected to another controller in the cluster query

Response Body

element: list
media types: application/xml
application/json

A list of Nodes org.opendaylight.controller.sal.core.Node 


Example:

Request URL:
http://localhost:8080/controller/nb/v2/connectionmanager/nodes?controller=1.1.1.1

Response body in XML:
<list>
<node>
<id>00:00:00:00:00:00:00:52</id>
<type>OF</type>
</node>
<node>
<id>00:00:00:00:00:00:00:3e</id>
<type>OF</type>
</node>
</list>

Response body in JSON:
{
"node": [
{
"type": "OF",
"id": "00:00:00:00:00:00:00:52"
},
{
"type": "OF",
"id": "00:00:00:00:00:00:00:3e"
}
]
}

Status Codes

code description
401 User not authorized to perform this operation
406 Invalid Controller IP Address passed.
503 Connection Manager Service not available

/node/{nodeId}/address/{ipAddress}/port/{port}

Mount Point: /controller/nb/v2/connectionmanager/node/{nodeId}/address/{ipAddress}/port/{port}

PUT

If a Network Configuration Service needs a Management Connection and if the Node Type is unknown, use this REST api to connect to the management session. 


Example :

Request :
PUT http://localhost:8080/controller/nb/v2/connectionmanager/node/mgmt1/address/1.1.1.1/port/6634

Response :
Node :
xml :
<node>
<id>mgmt1</id>
<type>STUB</type>
</node>

json:
{"id": "mgmt1","type": "STUB"}

Parameters

name description type default
nodeId User-Defined name of the node to connect with. This can be any alpha numeric value path
ipAddress IP Address of the Node to connect with. path
port Layer4 Port of the management session to connect with. path

Response Body

element: node
media types: application/xml
application/json

Node If the connection is successful, HTTP 404 otherwise.

Status Codes

code description
401 User not authorized to perform this operation
404 Could not connect to the Node with the specified parameters
406 Invalid IP Address or Port parameter passed.
503 Connection Manager Service not available

Response Body

element: node

application/json
